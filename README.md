# MITERAS CLI

An (**unofficial**) command-line tool for [MITERAS](https://www.persol-pt.co.jp/miteras/).

## Usage

### login

Save credentials using the `login` command.

```console
$ miteras login
Try logging in to MITERAS.

Org: A123456
Username: sinsoku
Password: ********

Login successful.
```

### clock-in / clock-out

```console
$ miteras clock-in
$ miteras clock-out
```
